# Greenwave Systems

This app adds support for devices made by [Greenwave Systems](http://www.greenwavesystems.com).

## Supported devices with most common parameters:
* Greenwave powernode-1
* Greenwave powernode-6

## Supported Languages:
* English
* Dutch
